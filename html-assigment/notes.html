Five HTML tags

    1. H1

      2. The H1 tag, usually the biggest and most noticeable heading, is the primary heading on a web page. Its purpose is to define the page title. To make the H1 stand out from other headings like H2 or H3, websites often utilize CSS.

    2. < P >

      3. A paragraph in an HTML document consists of various content, such as text, images, or form fields. It is enclosed by an opening tag and a closing tag. These paragraphs are part of the block-level elements category.

    3. < a >

      4. The < IMG > tag is utilized to insert small graphics or icons into an HTML document. It is not meant for inserting other HTML text. If you need to include larger figures with captions and text flow, use the FIG element..

    4. < img >

      5. The tag is utilized to insert graphics or icons into an HTML document. It is not meant for inserting other HTML text. EXAMPLE < img src="image.jpg" alt="Description of the image">

    5. < UL >

      6. The < ul > tag defines an unordered (bulleted) list. Use the < ul > tag together with the > tag to create unordered lists. Tip: Use CSS to style lists. Tip: For ordered lists, use the < ol > strong> tag.
